Prueba - GitHub Repository

Lista de pasos

  1. Crear o muévete hacia la carpeta deseada
  2. Abre la consola y ejecuta el siguiente comando para crear un repositorio git: git init
  3. Crea los siguientes archivos:
    1. .gitignore
    2. .editorconfig
    3. README.md
  4. Para crear el archivo package.json utilizamos el siguiente comando: npm init -y
  5. Añadimos los archivos creados al staging area del repositorio utilizando los siguientes comandos:
    1. Para añadir los archivos: git add .
    2. Para capturar los cambios del proyecto: git commit -m "Initial commit"
  6. Para publicar la rama del proyecto, nos dirigimos al icono de rama que esta en la barra de la izquierda de Visual Code
    • Clickeamos "Public Branch" y se abrirá una ventana
    • Selecciona la opción "public repository"
  7. Abrimos GitHub para seleccionar el repositorio creado
    1. Vamos a Settings y de ahi a Pages
    2. Buscamos el brand y seleccionamos main
    3. Arriba saldrá un recuadro que hará referencia al website creado
